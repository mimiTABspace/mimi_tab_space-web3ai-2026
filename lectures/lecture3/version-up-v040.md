## 主要新機能

### カレンダー＋体調グラフページ（`/calendar`）

ナビバーの「ルート」を「カレンダー」に刷新し、新ページを新設。

**タブ1「カレンダー」**
- 月表示カレンダーで服薬記録・通院予定をまとめて確認
- 服薬: 飲んだ日に✅、飲めなかった過去日に❌アイコン
- 通院予定: 🏥アイコン表示
- 日付タップで詳細ポップアップ（病院名・薬の名前・時刻）
- 前月/翌月ナビゲーション

**タブ2「体調グラフ」**
- 気分・疲労度の推移を折れ線グラフで表示
- 1週間 / 1ヶ月の切り替え
- 同日複数記録は平均値で表示
- ライブラリ: `recharts`
- 既存の `health_logs` テーブルをそのまま利用

---

### Care Route をホームの Main Quest に統合

- ナビバーから「ルート」タブを削除
- ホームの Main Quest セクションに QuestCard として「Care Route」を常時表示
  - 治療中のルートがある場合: 「治療中: N件」
  - ない場合: 「治療中のルートなし」

---

### 通院予定の編集・削除

予定タブの各通院カードに操作ボタンを追加。

| ボタン | 動作 |
|--------|------|
| 受診完了 | 受診結果・次回予定を入力して完了記録 |
| 📅 日時・目的変更 | 受診日時と目的をその場で変更 |
| 🗑 削除 | 確認ダイアログ付きで予定を削除 |

---

### ショップ機能（`/shop`）

コインを使ってアイテム購入・くじ・銀行預け入れができるページを新設。

**アイテム購入タブ**
- おやつ×1（⭐）・×3（⭐⭐）・×10（⭐⭐⭐）を販売
- 種族限定おやつ（⭐⭐）: アンロック済み種族のみ表示
- 毎月1日・15日はセール（20%OFF）
- ホームから直接遷移できる導線カード（コイン残高付き）を追加

**くじタブ**
- 1回30コイン / 10連300コイン（★★以上1枚確定）
- カードフリップ演出（タップでめくる・★★★/★★★★は特別エフェクト）
- 称号ダブり時はレアリティに応じたコイン自動還元
- くじ履歴（過去10回分）
- 初回★★★★で「天運の覇者」称号付与
- ※本番は「仕入れ旅中」表示（デバッグモードのみ使用可）

**銀行タブ**
- コインを預けて毎日の連続記録達成時に利息付与
- 利率は全ユーザー共通（`seededRate` による決定論的生成）
- 利率グラフ（実績＝実線、予報＝破線）
- レベルに応じた利率予報精度（Lv.10〜100）
- 残高推移グラフ

**サブページ**
- `/shop/inventory`: 所持アイテム一覧
- `/shop/titles`: 称号一覧

**称号**
- 🛒 初来店の客（ショップ初使用）
- 👑 天運の覇者（初回★★★★大当たり）

## 機能改善

### 家族共有メンバーへのニックネーム設定

家族共有中のメンバーにニックネームを設定できるようになった。

**背景**
- これまで体調・服薬・通院ページで共有メンバーがメールアドレスで表示されており、誰のデータかわかりにくかった

**変更内容**
- 設定ページの「家族共有」セクションで各メンバーに名前をつけられるようになった
  - 例: 「お母さん」「パパ」など自由に設定
  - 設定済みの場合は「変更」ボタンで修正可能
  - 未設定の場合は従来通りメールアドレスを表示
- ニックネームは体調・服薬・通院ページのメンバー表示にも反映される

---

### 銀行機能の改善（2026-06-20）

**利息の自動付与**
- 体調記録（初回保存）時に朝分の利息を自動付与。トーストに `💰 利息+XXコイン！` を追記
- 金曜16時以降にショップページを開いた際、夕分の利息を自動付与
- `giveInterest(userId, period)` を `lib/supabase.ts` に追加。重複付与防止チェック付き

**残高推移の視認性向上**
- 残高推移にグラフ/リスト切り替えボタンを追加
- リスト表示: 日付・種別・増減コイン・残高を一覧表示。利息行に利率（%）も表示
- 利息ドットの色をグラフ線と同じ茶色 → ゴールド（`#f59e0b`）に変更して視認性向上
- 凡例を色付きドット付きに改善

**金曜の利率2点表示**
- 金曜の利率グラフを朝・夕の2点表示に対応
- 今日が金曜で16時前の場合、夕のエントリは非表示
- 朝の利率は `seededRate` で再計算するため、DB上書き後も正確な値を表示

**ホームのバナー通知**
- 金曜16時以降にホームを開くと「🏦 金曜夕の利率に更新！」バナーを表示
- 「銀行へ」ボタンでショップページに遷移、「✕」で閉じられる

## 技術メモ

- カレンダーの日付判定は `YYYY-MM-DD` 文字列の前方一致でフィルタ
- 体調グラフは日付ごとに `mood` / `fatigue` を平均化して描画
- 日時変更モーダルは `datetime-local` 入力値をローカルタイムオフセット補正してから ISO 文字列に変換
- `recharts` を依存関係に追加
- TypeScript の `ignoreBuildErrors: true` 設定によりimport漏れがビルド時に検出されなかった（要注意）
- Supabase の `family_shares` テーブルに `nickname text` カラムを追加
- 設定ページでインライン編集UI（保存・キャンセル）を実装
- 体調・通院ページは `_ownerEmail` に `nickname || member_email` を格納
- 服薬ページはセクション見出しで `share.nickname || share.member_email` を表示
- `seededRate(key)`: 決定論的ハッシュで全ユーザー共通利率を生成（Math.randomを使わない）
- アカウント削除Edge Functionはユーザー自身のJWTでデータ削除し、service_roleはauth削除のみに使用
- `toJSTDateStr(date)`: `Intl.DateTimeFormat("sv-SE", { timeZone: "Asia/Tokyo" })` を使い `YYYY-MM-DD` 形式のJST日付文字列を返すヘルパー。`toISOString()` はUTCを返すため銀行・ガチャの日付判定に使用禁止
- `giveInterest(userId, period)`: 朝（4時〜16時）・夕（16時〜翌4時）の期間で `bank_interest_logs` を確認し重複付与を防止。`coin_bank` と `bank_interest_logs` を同時更新
